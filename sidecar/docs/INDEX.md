## Mechanism Taxonomy

1. **Foundation** (a, b): Core PSN infrastructure -- ingestion, storage, recall
2. **Validation** (c): Compass validation, inflection point detection
3. **Injection** (e, f, h): Direct parameter/neuron injection into LLM weights
4. **Fusion** (g): Network merge attempt (invalid -- catastrophic)
5. **Parallel Sidecar** (i, j): Independent network running alongside LLM, merged at output
6. **ASN Steering** (l, m, n, o, q): Associative network biasing LLM generation
7. **Contaminated** (d, p): Results invalidated by experimental error
8. **Cage Analysis** (k): Probing whether PSN neurons are "caged" by LLM context
9. **Context Sidecar** (r): Hopfield recall for agent context (operational, not research)
10. **Static Sidecar** (s): Frozen knowledge capsule injected at inference time

## True Sidecars (the convergent pattern)

The arc converges on 4 true sidecar experiments:
- **sidecar-i** (MIRROR D 0.5B): First proof that parallel architecture works
- **sidecar-j** (MIRROR D 3B): Scaled validation at 3B parameters
- **sidecar-n** (Shannon Son G): ASN dark knowledge as real drive signal
- **sidecar-s** (V4 static): Production-path static knowledge capsule
